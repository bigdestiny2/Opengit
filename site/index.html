What it actually is

Drop-in git

opengit://<key> URLs work with stock git clone, git push, git fetch. The git-remote-opengit helper bridges to a per-repo bare-git shadow that git upload-pack and git receive-pack drive directly. No new VCS.

Owner sleeps, repo stays online

opengit-mirror (public) and opengit-relay (private, blind-encrypted) keep your repo reachable when your laptop is closed. Anyone can run a relay — or use the verified multi-region HiveRelay endpoints.

Crypto-authored issues + PRs

Issues and PRs are Autobase-backed thread systems. Every comment, review, merge is signed by an Ed25519 identity. Apply rules enforce who can close, merge, label. Signatures verify on every replica.

Browseable from your phone

opengit pages publish renders the repo's HEAD into a static-HTML Hyperdrive. Open hyper://<key>/ in PearBrowser on iOS — no Opengit-specific code on the browser side.

No telemetry. Ever.

Every binary makes only the network calls required by your stated operations. No analytics, no phone-home, no auto-update. The decentralization audit documents how, and the license check enforces no AGPL-by-default.

Apache-2.0 everywhere it matters

Six of seven packages are Apache-2.0 end-to-end. The seventh (opengit-relay) is Apache-2.0 by default; AGPL-3.0 only if you opt into --use-hiverelay's AGPL-licensed transport. Verified by scripts/check-licenses.js.

The 30-second tour

Push a public repo

npm install -g opengit-cli git-remote-opengit
opengit identity init
opengit init my-project
git remote add p2p opengit://$(opengit info my-project | grep ^key | awk '{print $2}')
git push p2p main

Render it for PearBrowser

opengit pages publish my-project
# → hyper://<driveKey>/

opengit pages watch my-project
# auto-republish on every push

Open an issue (signed)

opengit issue open my-project "Bug" --body "details"
opengit issue list my-project
opengit issue show my-project <issueId>

Search across indexers

# run an indexer (allowlist-only)
opengit-indexer --repo my-project &

# fan out queries — multiple indexers in parallel
node -e "const { OpengitForge } = require('opengit-core');
const { fanOutQuery } = require('opengit-indexer/query');
// ..."
